Background

The entire site is a web app written in python. Currently it is using python 2.7 (I would rather have it in python 3 but there is some problems with the utf-8 encoding, if you can fix it everything should be compatible with newer versions of python).

The entire site source code (minus datasets) is available at this github repository:

github.com/LuisSoares/H3K4_webapp

To run the site locally just you can just clone the repository and launch web_site.py. The code is not really commented but with minimal python knowledge you will be able to costumize the site to your datasets and preferences. If you want to deploy the web site yourself make sure to change the default JavaScript in each app since it currently links to my google analytics.There is also a helper script called Data_load.py that creates the hd5 files with the datasets that are used in web site. I was able to successfully deploy the website in both heroku and pythonanywhere for global access, if you need instructions contact me.

Dependencies

The website is built with the spyre frame work that is built on top of cherrypy. There are several dependencies that must be installed (they are all available with pip) to run the script. The dependencies can be infered by the imports of each .py file but the list is:

spyre:
build the framework
cherrypy:
To serve the the website (I think it is installed with spyre)
h5py:
To create and access the datasets
numpy:
To deal with the datasets data
matplotlib:
To draw the plots in the Gene plotter
intermine:
To link the gene names to gene codes using SGD data
pandas:
To draw the tables in the Gene plotter
bokeh:
To draw the plots in Comparisons
